# Kinesis Data Analytics — Operations

## List Applications
aws kinesisanalyticsv2 list-applications

## Describe Application
aws kinesisanalyticsv2 describe-application   --application-name <name>

## Start Application
aws kinesisanalyticsv2 start-application   --application-name <name>

## Stop Application
aws kinesisanalyticsv2 stop-application   --application-name <name>

## Monitor Metrics (CloudWatch)
- InputProcessingRate
- OutputProcessingRate
- MillisBehindLatest
